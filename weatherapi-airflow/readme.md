# 🌦️ Automated Weather Data Pipeline

## 📌 Overview
This project implements an automated data pipeline to fetch daily weather data from a REST API, process it, and store it in a relational database.

## 🚀 Features
- Fetches real-time weather data using API
- Data transformation using Python (Pandas)
- Stores data in MySQL
- Automated scheduling using Apache Airflow

## 🛠 Tech Stack
- Python
- Pandas
- MySQL
- Apache Airflow
- REST API

## 🔄 Workflow
1. Fetch data from API
2. Transform and clean data
3. Store in database
4. Schedule using Airflow

## ▶️ How to Run
1. Add API key in config/config.py
2. Setup MySQL database
3. Run script manually or via Airflow DAG

## 📊 Future Enhancements
- Cloud deployment (Azure/AWS)
